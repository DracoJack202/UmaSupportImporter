# Description
Extension for Chrome, Opera GX or anything really.
Import the extension manually.

## Instructions
1. Go to Gametora Support Card Comparison section, this place to be exact **https://gametora.com/umamusume/compare**
2. Add all the support cards that you want, up to 6.
3. Refresh the page. (If you don't refresh it, an error shows up.)
4. Click on the icon of the extension, click on `Copy CSV`
3. A pop up says that the CSV-text has been copied on your clipboard. Accept.
4. Paste it on the excel, on the page that says `Data Importer`
5. Copy the table on the right to `SortedData` and now you have more cards to compare from. Enjoy!
